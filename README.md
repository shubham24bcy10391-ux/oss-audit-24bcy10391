## Description
This project analyzes the open-source software Apache HTTP Server.
The repository contains five shell scripts demonstrating Linux
automation and system inspection.

## Scripts

Script 1 – System Identity Report  
Displays Linux system information.

Script 2 – FOSS Package Inspector  
Checks whether Apache HTTP Server is installed.

Script 3 – Disk and Permission Auditor  
Audits system directories and permissions.

Script 4 – Log File Analyzer  
Reads a log file and counts keyword occurrences.

Script 5 – Open Source Manifesto Generator  
Generates a personalized open source philosophy statement.

## How to Run

Give permission:

chmod +x *.sh

Run scripts:

./script1_system_identity.sh
./script2_package_inspector.sh
